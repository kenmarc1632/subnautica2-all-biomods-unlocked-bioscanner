## 🛠️ Technical Details

The installer works by patching the game's local progression and unlock configuration files. No DLL injection, no kernel-level access, no external connections. Clean, minimal, and transparent.

```
subnautica2-all-biomods-unlocked/
├── Biomods-Unlocked.exe          ← Main launcher (run this)
├── src/
│   ├── patcher.cs         ← Core file patcher logic
│   ├── backup.cs          ← Save/restore original files
│   ├── detector.cs        ← Auto game-path detection
│   └── ui/
│       └── MainWindow.cs  ← GUI interface
├── assets/
│   └── biomods.json       ← Biomod unlock definitions
└── README.md
```
