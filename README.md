## 🔍 Troubleshooting

**The installer cannot find the game folder.**
If the automatic search fails, click the Browse button. Navigate manually to the folder where you installed the game. For Steam users, this is typically located within `Program Files (x86)\Steam\steamapps\common\Subnautica2`. 

**The game crashes on startup.**
Verify your game files through Steam or the Epic Games Launcher. Corrupted game data causes crashes. After a verification, run this mod installer again.

**Biomods do not appear in the menu.**
Ensure you are using the latest version of the game. If you play on an outdated version, the mod may fail to inject the correct data. Check the GitHub page frequently for updates that match new game patches.
